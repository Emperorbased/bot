# Telegram Appeal Bot

Telegram бот для обжалования наказаний и подачи жалоб на персонал.

## Установка локально

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python bot.py
```

## Деплой на Render

1. Загрузите код на GitHub
2. Создайте новый Web Service на Render.com
3. Подключите ваш GitHub репозиторий
4. Render автоматически обнаружит Python и установит зависимости

## Функции

- `/start` - Главное меню
- `/addadmin` - Добавить администратора (только для главных админов)
- Обжалование наказаний
- Жалобы на персонал
- Ответы администраторов на жалобы
